# ARHW5
This project is a headmounted display (HMD) on an Oculus Quest 3 project build in Unity with the Meta SDK.
This app allows users to passthrough to overlay a virtual map and objects onto the real world. Users can interact with the ball and moveable blocks. 

# Features #
- Passthrough AR environment
- Push and movement on objects
  
## Packages ## 
- Meta XR Core SDK
- Meta XR Interaction SDK
